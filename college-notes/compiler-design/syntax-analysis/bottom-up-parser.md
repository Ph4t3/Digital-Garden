# Bottom-Up Parser

## LR\(0\)

\(0\) means no lookahead.

Shift input symbol to stack  
If possible Reduce  
Repeat

{% hint style="info" %}
Will have problems if wrong prodcution is reduced. Solved using lookaheads.
{% endhint %}

> S → cAd  
> A → a
